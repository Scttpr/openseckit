# Secure Prompts

AI prompts for the Secure phase of OpenSecKit.

## Available Prompts

| Prompt | Command | Description |
|--------|---------|-------------|
| `osk-secure-specify.md` | `/osk-secure-specify` | Create security specification with threats and requirements |
| `osk-secure-clarify.md` | `/osk-secure-clarify` | Q&A workflow to clarify ambiguities |
| `osk-secure-plan.md` | `/osk-secure-plan` | Generate security implementation plan |
| `osk-secure-tasks.md` | `/osk-secure-tasks` | Generate ordered tasks with dependencies |
| `osk-secure-implement.md` | `/osk-secure-implement` | Execute tasks and update risk register |

## Workflow

```
specify → clarify → plan → tasks → implement
```

See the [Secure Phase documentation](../README.md) for detailed workflow.
