# Secure Schemas

YAML schemas defining data structures for the Secure phase.

## Schemas

| Schema | Description |
|--------|-------------|
| `security-spec.yaml` | Security specification structure |
| `risks.yaml` | Risk register format |

## Usage

These schemas define the structure of YAML files generated by the secure workflow:
- `.osk/specs/<feature>/spec.yaml`
- `.osk/specs/<feature>/risks.yaml`

Schemas follow JSON Schema draft 2020-12 format.
